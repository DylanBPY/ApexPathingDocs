# Localizer Constants
---
Choose your localizer type below and follow the steps to configure it.

<Tabs items={['goBILDA Pinpoint', 'SparkFun OTOS', 'Three Pod', 'Two Pod', "Octoquad", 'Drive Encoders']}>
  <Tabs.Tab>
    <Steps>
    ### Replace `localizerConfig()`
    ```java
    @Override
    public BaseLocalizerConfig<?> localizerConfig() {
        return new Pinpoint.Config()
                .setName("pinpoint")
                .setEncoderResolution(Pinpoint.GoBildaPods.goBILDA_4_BAR_POD)
                .setEncoderDirections(Pinpoint.EncoderDirection.FORWARD, Pinpoint.EncoderDirection.FORWARD)
                .setOffsets(0, 0, DistUnit.IN);
    }
    ```
    ### Name
    Replace `"pinpoint"` with the name of your Pinpoint computer in the hardware configuration. 

    ### Resolution
    If you are using goBILDA odometry pods, you can use the predefined resolutions `Pinpoint.GoBildaPods.goBILDA_4_BAR_POD`
    or `Pinpoint.GoBildaPods.goBILDA_SWINGARM_POD`. If you are using a custom odometry pod, the input should be a [Dist](/docs/classes/dist/)

    ### Directions and offsets
    Download and run the `OdometryTuner` OpMode from the [OpModes](/docs/opmodes/) page. This will allow you to determine the 
    correct encoder directions and offsets for your robot. You can manually tune these values if you'd like.
    </Steps>
  </Tabs.Tab>
  <Tabs.Tab>
    <Steps>
    ### Replace `localizerConfig()`
    ```java
    @Override
    public BaseLocalizerConfig<?> localizerConfig() {
        return new OTOS.Config()
                .setName("otos")
                .setOffset(new Pose(Vector.of(0, 0, DistUnit.IN), Angle.fromDeg(0)))
                .setLinearScalar(0)
                .setAngularScalar(0);
    }
    ```
    ### Name
    Replace `"otos"` with the name of your OTOS sensor in the hardware configuration. 

    ### Scalars and offsets
    Download and run the `OdometryTuner` OpMode from the [OpModes](/docs/opmodes/) page. This will allow you to determine the 
    correct scalars and offsets for your robot. You can manually tune these values if you'd like.
    </Steps>
  </Tabs.Tab>
